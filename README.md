# 海关总署数据清洗工具

清洗海关CSV，支持多文件合并、单位转换、多级排序、自动删列。

## 功能

- 拖拽文件或文件夹添加CSV
- 列名可映射（默认：数据年月、商品编码、商品名称、贸易伙伴名称、第一数量、人民币）
- 数量单位：千克→吨/克/磅
- 金额单位：人民币→美元/欧元/英镑（自定义汇率）
- 数量、金额分别设小数位数
- 商品编码过滤（包含/排除，|分隔）
- 去重，缺失值填0或删行
- 清洗后自动删除原始数量/金额列，删除计量单位、第二数量、Unnamed等冗余列
- 多级排序（最多三级，动态增删，升降序）
- 多选删除任意列
- 保存为CSV（UTF-8-BOM）

## 运行

- 直接运行 exe
- 或 `python customs_cleaner.py`（需安装 pandas, PyQt5, chardet）
