# 海关数据清洗工具

## 功能
- 支持多CSV文件拖拽/选择
- 自定义列名映射（适配不同格式）
- 数量单位转换（千克/吨/克/磅）
- 金额单位转换（人民币/美元/欧元/英镑）+ 手动汇率
- 商品编码过滤（包含/排除，支持 `|` 多值）
- 清洗选项：去重、缺失值填充0或删除行
- 输出：合并为一个CSV 或 分别保存每个文件
- 统计：清洗前后行数、数量/金额总计与均值、商品编码金额TOP10

## 运行环境
- Windows 7/10/11
- 无需安装Python（直接运行exe）

## 使用方法
1. 双击 `CustomsCleaner.exe` 启动工具
2. 将CSV文件拖入窗口，或点击“添加CSV”
3. 如果列名不是默认值（数据年月、商品编码...），在“列名映射”区域修改
4. 选择单位、汇率、过滤规则
5. 点击“开始清洗并统计”
6. 查看统计结果，点击“保存合并数据”或“分别保存每个文件”

## 注意事项
- 原始数据中的数量列必须为千克，金额列必须为人民币元
- 过滤条件中的编码支持部分匹配（如 `2530` 可匹配 `25309099`）
- 多值用英文竖线分隔，例如：`25309099|260111`
- 输出的CSV使用 UTF-8-BOM 编码，Excel可直接打开
